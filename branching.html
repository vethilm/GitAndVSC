Branching

< Previous Next >

A branch is a version of a repository that is saved parallel to the main version. This allows you to work without impacting the main branch and is often used to develop new features or to fix bugs. Branches can be merged to add changes from one branch to another.

Creating a branch

In VS Code, you can create a branch through the Source Control Panel ( ). Click the 3 dots in the top right and select Branch, then Create Branch. It will then prompt you for a branch name, where name it whatever you want.
